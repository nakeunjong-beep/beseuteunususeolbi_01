# 📸 실제 시공 현장 사진 업로드 가이드 (Image Upload Guide)

안녕하세요! 베스트누수설비의 실제 시공 현장 사진을 홈페이지에 반영하려면, **왼쪽의 파일 탐색기**를 통해 이 폴더(`public/assets/`)에 아래 파일명으로 이미지를 업로드해 주세요.

## 📁 권장 파일명 및 매칭 안내
1. **누수탐지 & 누수공사**: `leak.jpg`
2. **하수구막힘 & 하수구 고압세척**: `sewer.jpg`
3. **싱크대막힘**: `sink.jpg`
4. **수전교체**: `faucet.jpg`
5. **세면대교체 & 변기교체**: `toilet.jpg`
6. **해빙 & 언수도녹임**: `thawing.jpg`

---

### 💡 참고 사항 (Tip)
- 파일 형식은 `.jpg`, `.jpeg`, `.png`, `.webp` 모두 지원합니다.
- 사진을 업로드하시면 별도의 코드 수정 없이 **즉시 홈페이지에 실제 공사 사진으로 교체되어 반영**됩니다!
- 사진이 업로드되지 않은 상태에서는 깔끔한 기본 이미지(Unsplash)가 **자동으로 안전하게 대체 표시(Fallback)**됩니다.
